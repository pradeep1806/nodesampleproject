## nodesampleproject

Same project made using node js to showcase best coding practices.

## Horse Requirement

Backend api for horse buying selling.

## Usage

Edit configurations in config/config.env

## Install Dependencies

npm install

# Run in dev mode

npm run dev

# Run in prod mode

npm start
